# Apex Plugins
This repo contains my own plugins for Revenge.

# How to install?
Paste a plugin URL into the Plugins page of Revenge, following a basic format of:
```
https://rillonDodgers.github.io/Apex-Plugins/<PLUGIN_NAME>
```
## Or just copy plugin links from here!
- [Message Scheduler](https://rillonDodgers.github.io/Apex-Plugins/messageScheduler/) - schedule messages to be sent at specific time or after a delay! (port from Vencore) **NOTE: As mentioned by one of Revenge mods, this plugin MAY FALL under selfbotting along with the og one from vencord/vencore, thus won't be proxied and added to offical list, USE AT YOUR OWN RISK!!!**
- [More Alts](https://rillonDodgers.github.io/Apex-Plugins/MoreAlts/) - use more than 5 alts at Discord and easily switch between them from chat input. [For FAQs go here](https://github.com/RillonDodgers/Apex-Plugins/wiki/MoreAlts)
- [Read All](https://rillonDodgers.github.io/Apex-Plugins/ReadAll) - Read all server and DM notifications with single command - /readall (idea from btmc727)
- [BetterNSFWGateBypass](https://rillonDodgers.github.io/Apex-Plugins/nsfwbypass) - Allows you to access NSFW channels without setting your age[.](https://m.youtube.com/watch?v=xvFZjo5PgG0) [Note](https://github.com/RillonDodgers/Apex-Plugins/tree/master/docs/nsfwbypass)
